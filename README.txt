CONTENUTO DELLA CARTELLA

I file "crypt_mono.py" e "crypt_poli.py" sono codici per crittografare un testo di input con i metodi mono-alfabetico e poli-alfabetico rispettivamente; il file criptato viene stampato su un file chiamato "cripted.txt" e la chiave viene stampata sul terminale.

Il file "crypto.py" contiene oggetti utili per la crittografia e viene incluso in tutti gli altri codici, in particolare contiene i metodi per criptare e decriptare testi e le liste contenenti i bigrafi, trigrafi e parole più frequenti della lingua inglese.

Il file "crypto_test.py" contiene il test dei metodi scritti in "crypto.py"

I file chiamati "Exercise" contengono il codice che risolve l'esercizio assegnato dal tema d'esame con il numero corrispondente.

Ho incluso anche 3 testi formato txt che ho usato per testare i miei codici.

Tutti i codici prevedono di essere eseguiti con argomenti aggiuntivi da linea di comando, e sono documentati tramite il comando "python3 <nome_file> --help".

NB: lo spazio fa sempre parte dell'alfabeto, quindi è necessario che sia presente nel testo originale
